# Introducción a la Computación en la Nube con GitHub

## Descripción del proyecto
Este repositorio forma parte de una práctica introductoria sobre computación en la nube.  
El objetivo del proyecto es familiarizarse con el uso de GitHub como una herramienta cloud para el almacenamiento, gestión y control de versiones de archivos.

---

## ¿Qué es la computación en la nube?
La computación en la nube es un modelo que permite acceder a recursos informáticos como almacenamiento, aplicaciones y servicios a través de internet, sin necesidad de depender de un dispositivo físico específico.

---

## ¿Qué es Git?
Git es un sistema de control de versiones que permite registrar y gestionar los cambios realizados en archivos y proyectos a lo largo del tiempo, facilitando el trabajo organizado y seguro.

---

## ¿Qué es GitHub?
GitHub es una plataforma basada en la nube que utiliza Git para alojar repositorios, permitiendo la colaboración, el control de versiones y el acceso remoto a proyectos desde cualquier lugar.

---

## Objetivo de la práctica
- Comprender los conceptos básicos de computación en la nube.
- Utilizar GitHub como entorno de trabajo en la nube.
- Aplicar control de versiones mediante commits.
- Organizar correctamente un repositorio.
